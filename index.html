Estructuras condicionales 2

  1. Introducción:

    En una estructura condicional múltiple, se evalúa una variable y se la compara con múltiples valores llamados casos. Si la variable es igual a alguno de los mencionados valores, es decir, si se cumple alguno de los casos, el algoritmo seleccionará el camino asociado a dicho caso y se realizarán todas las acciones dentro de él. A continuación, se ilustra el diagrama de flujo de una estructura condicional múltiple:

    Diagrama de flujo de la estructura switch.

    En este ejemplo, la variable de la condición es comparada con los valores A, B y C. Si la variable de la condición coincide con alguno de los valores, se realizarán las acciones asociadas a ese camino. Esto quiere decir que, si la variable de la condición coincide con el valor A, se realizará la Acción 1, si coincide con el valor B, se realizará la Acción 2, y si coincide con el valor C, se realizará la Acción 3. 

    
    switch( variable )
        {
            case 'A':
                accion1;
            break;
            case 'B':
                accion2;
            break;
            case 'C':
                accion3;
            break;
            default:
                accion4;
            break;
        }

    Sintaxis en C de un switch.

    La estructura switch posee además un espacio default para realizar una acción en la situación de que la variable no se corresponda con ninguno de los casos comprendidos.

    Ejemplo

    Realizar un programa en C que imprima en pantalla en forma textual los números enteros entre el 1 y el 3, ambos inclusive, utilizando la estructura switch

    Solución: 
    
    #include<stdio.h>
    
    int main(){
        int numero;
        scanf("%i",&numero);
        switch(numero)
            {
                case 1:
                    printf("Uno.");
                break;
                case 2:
                    printf("Dos.");
                break;
                case 3:
                    printf("Tres.");
                break;
                default:
                    printf("Numero incorrecto");
                break;
            }
        return 0;
    }

  2. Estructura condicional múltiple else if sino si

    En una estructura condicional múltiple sino si (conocida por su nombre en inglés else if), se utiliza para evaluar múltiples condiciones y ejecutar diferentes acciones según el resultado de cada evaluación. A continuación, se ilustra el diagrama de flujo de una estructura condicional múltiple sino si.

    Diagrama de flujo estructura sino si.

    En esta estructura, se evalúa la condición 1, y en caso de que el resultado sea verdadero, se realizará la acción 1. Pero en caso de que el resultado sea falso, se evaluará una segunda condición (la condición 2), y si esta última es verdadera se realizará la acción 2. Finalmente, si no se cumple ninguna de las condiciones, se realizará la acción 3. En este ejemplo, sólo se evalúan dos condiciones. pero este tipo de estructura puede evaluar todas las condiciones que sean necesarios.

    Sintaxis 
    
    if(condicion1)
        {
            accion1;
        }
        else if(condicion2)
            {
                accion2;
            }
            else
            {
                accion3;
            }

    Sintaxis en C de un sino si.

    A continuación, se muestra un ejemplo de cómo se aplica una estructura sino si.

    Ejemplo

    Realizar un programa en C que decida si un número ingresado por teclado es mayor, menor o igual a cero, utilizando la estructura sino si.

    Solución 
    
    #include<stdio.h>
    
    int main()
        {
            int a;
            printf("Introduce un número: ");
            scanf("%i",&a);
            if(a > 0)
                {
                    printf("El número es positivo. \n");
                }
                else if(a < 0)
                    {
                        printf("El número es negativo. \n");
                    }
                    else
                    {
                        printf("El número es cero. \n");
                    }
        return 0;
        }
